Sandbox RESTful API Documentation

Start a new Sandbox

GET /start/image_name - Returns a socket to a VM to telnet into.

Stop an existing Sandbox

GET /stop/socket - Stops a socket with an appropriate HTTP response code (e.g., 200 OK, 404 Not Found).

Check if a Sandbox is running

GET /check/socket - Returns a JSON object with the image name or an appropriate HTTP response code. (e.g., 404 Not Found).

Reset a running Sandbox

GET /clean/socket - Returns a JSON object with the image name or an appropriate HTTP response code. (e.g., 404 Not Found).
